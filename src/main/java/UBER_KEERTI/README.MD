# Functional Requirements

## Driver

1. Create Profile
2. Show Availibility
3. Accept/Reject Ride
4. Navigate to Pickup
5. Start Ride
6. Navigate to Drop
7. End Ride
8. Payment


## Rider

1. Create Profile
2. show cabs nearby
3. Add start/End Location
4. Show Estimation of time and price
5. Book Cab
6. Make Payment
7. Rate the driver
